# Jacco-Mols_Web-DD_HCD

## Week 1

### Maandag 30 Maart
Vandaag zijn we begonen met de kick-off van het vak HCD. Hier hebben we een korte uitleg gekregen over de opdracht. De opdracht is om een web-applicatie te maken voor een specifiek persoon. Mijn persoon is Darice, zij wilt graag een website waarin je een film of podcast kan zien. Omdat Darice doof is, wilt zij graag aanduidingen van omgevingsgeluiden, emoties en soorten stemmen die rond spelen in de content.

Ik heb voor een podcaast gekozen, omdat ik hier snel leuke iedeeën voor kon bedenken. Ik had wat schetsen gemaakt (FOTO VAN SCHETSEN??????) over de verschillende functionaliteiten die ik kan toevoegen om tot Darice haar wensen te voldoen. Mijn eerste concept waar ik vandaag mee bezig was, was een kleine combi van wat ideeën. Ik ben na veel onderzoeken naar wat mogelijk is op het internet gekomen op een site die een transcript uitschrijft vanuit een gekozen podcast. Deze transcript kan ik weer omzetten naar een WEBVTT format wat weer toegankelijk is voor het internet. Dit bestand past de text via timestamps aan in de placeholder. Ik heb nu dus een soort voorlees programma voor de podcast, ik hoop alleen nog veel meer leuke functies te kunnen toevoegen aan het programma zoals indicaties van stem-volume, emoties in stemmen en achtergrondgeluiden.
